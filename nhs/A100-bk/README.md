# A100 原始版本备份

本目录保存生产环境脚本在加入“真实路径增量获取”和“MDE 评估”功能之前的原始版本，用于版本对照、问题排查和必要时恢复。

包含的主要文件：

- `ty_main_auto.py`：原生产主控脚本。
- `get_typhoon_data _no_gt.py`：原台风样本生成脚本。
- `infer_case_auto_plot.py`：原模型推理脚本。
- `utils.py`、`values.py`：推理相关辅助模块。

注意事项：

- 本目录仅作为备份，不应作为日常运行目录。
- 备份代码不包含真实路径获取、观测时次回溯和 MDE 增量计算功能。
- 如需恢复，请先对比当前 `nhs` 目录中的生产文件，确认不会覆盖后续业务修改。
